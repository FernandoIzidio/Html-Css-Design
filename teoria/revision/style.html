Cascading Style Sheet

Conhecido como Cascading Style Sheet, ou folha de estilo em cascata o CSS é uma linguagem de estilização que permite personalizar sites de forma consisa, e até mesmo criar animações profissionais

Extrutura

O CSS funciona ao lado do HTML, e é usado para atribuir propriedades de estilo a tags HTML, o CSS atribui propriedades de estilo a tags HTML por meio de seletores, as propriedades podem ser aplicadas por chave e valor property: value; sendo sempre delimitadas por prop: value;

Seletores

            tag, tag {} -Aplica propriedades as duas tags selecionadas

            * {} - Seleciona todas as tags do html

            tag - Seleciona todos as tags de valor tag

            .class - Seleciona uma tag por classe

            #id - Seleciona uma tag por identificador

            tag tag2 - Seleciona uma tag2 que esta dentro de tag, independente se tag2 esteja dentro de outra tag, ou seja esse seletor vai pegar os filhos e netos que corresponderem aos valores

            tag > tag2 - Pega apenas a tag2 que for diretamente filha de tag 

            .class.class2 - Seleciona apenas as tags que tiverem ambas as classes

            tag1+tag2 - Seleciona a primeira tag2 que estiver no mesmo escopo, ou estiver ao lado da tag1, ou seja vai selecionar a primeira tag2 irmã a tag1

            tag1~tag2 - Seleciona todas as tags2  diretamente irmãs a tag1

            [attr]{} - Seleciona apenas tags que tem determinado atributo

            [attr="value"] - Seleciona apenas tags que tiverem determinado apenas valor especificado no atributo

            [attr~="value"] - Seleciona apenas tags que tiverem pelo menos o valor no atributo

            [attr^="value"] - Seleciona apenas tags que começarem com determinado valor de atributo

            [attr$="value"] - Seleciona apenas tags que terminarem com os valores de atributos especificados

Unidades de medida

Absolute lenghts

Unidades de medidas de tamanhos absolutos tem valor fixo, e não mudam independente do tamanho dos outros elementos. Exemplos:

- px

Relative lenghts

Unidades de medidas de tamanhos relativos referênciam o tamanho de outras tags, ou seja o tamanho da tag pode mudar se o tamanho de outras mudarem. Exemplos:

                % - Referência a parent-box
                em - Referência a parent-box
                rem - Referência o font-size do root(tag html)
                vw - Referência a largura da viewport(largura do dispositivo)
                vh - Referência a altura da viewport(altura do dispositivo)
                fr - Referência o espaço em branco da parent box
                vmin - Referência o menor tamanho da viewport
                vmax - Referência o maior tamanho da viewport

Pseudo-Classes

Pseudo classes estão mais ligados ao estado das tags, existem varios tipos de pseudo classes para aplicar estilos, conforme o estado dos elementos, e as peseudo classes mais comuns: 

            -Hover - Aplica estilo quando o mouse passar por cima
            -link - Aplica estilo em link nunca visitado, ou botão nunca pressionado
            -visited - Aplica estilo em link visitado, ou botão pressionado
            -focus - Quando o elemento estiver em foco, ou seja quando o usuário estiver interagindo com um elemento esse estilo vai ser aplicado
            -active - Aplica estilo em elemento clicado
            -disabled - Se a tag estiver desabilitada, aplica estilo
            -enabled - Se a tag estiver habilitada, aplica estilo
            -checked - Se a tag estiver marcada em uma checkbox, aplica estilo
            -required - Aplica estilo em todas tags com atributo required
            -target - Aplica estilo em elemento, se seu id estiver na url da pagina
            - tag:not(#id/.classe/:estado) - Seleciona todas as tags que não tenham o id/classe/estado especificado

            - tag:empty - Seleciona elementos que não tem filhos
            - tag:nth-child(indice) - Seleciona todas as tags, que tiverem o estado de posição correspondente ao indice, ou seja a tag filha que estiver na posição indice
            - tag:nth-child(xn) - Seleciona os filhos de x em x
            - - tag:nth-child(xn + y) - Seleciona os filhos de x em x, começando a contar a partir de y
            - tag:nth-child(even) - Seleciona todas as tags, que tiverem o estado de posição par, ou seja todas as tags filhas que estiverem na posição par
            - tag:nth-child(odd) - Seleciona todas as tags, que tiverem o estado de posição impar, ou seja todas as tags filhas que estiverem na posição impar

            - tag:first-child - Seleciona tag que estiver com o estado de primeiro filho
            - tag:last-child - Seleciona tag que estvier com o estado de ultimo filho

Pseudo-Elementos

Normalmente são usados para criar tags, antes ou após outros tags, ou seja pseudo elementos estão mais relacionados com conteúdo. Exemplos

            - tag1::before - Cria nova tag antes da tag1
            - tag1::after - Cria nova tag depois de tag1
            - tag1::first-letter - Seleciona primeira letra do conteúdo de uma tag
            - tag::selection - Se a tag estiver sendo selecionada com o mouse para fazer control+c aplicar estilo
            - tag::first-line - Seleciona primeira linha de uma tag
            - tag::placeholder - Aplica estilo no placeholder de uma tag

Especificidade

Especificidade é a principal regra, ou regra que ocupa o topo da hierarquia na hora de aplicar um estilo em uma tag, quanto maior o nivel de especificidade de um elemento, mais força o estilo desse elemento ganha sobre outras regras de estilos

            id - seletores de maior especificidade, para sobrepor o estilo de um seletor de id, somente com outro seletor de id
            classes, pseudoclasses, atributos - tem mesmo valor de especificidade
            elementos, Pseudo-Elementos - tem mesma especificidade e estão no nivel mais baixo da hierarquia de especificidade, ou seja qualquer seletor pode sobrepor um seletor que seleciona por tag, ou pseudo-elemento

propriedades que terminam com !important são aplicadas independente da Especificidade ou de qualquer atributo, mas se dois seletores feitos de forma diferentes, mas que selecionam a mesma tag, usam !important em suas propriedades, a especificidade será usada como critério de desempate, hierarquia:

            1 - !important
            2 - style - atributo
            3 - id 
            4 - classes, pseudo-classes, atributos
            5 - elementos, pseudo-elementos

OBS: Atributo style, tem mais especificidade que um identificador/id

Herança

Determinadas propriedades de estilo podem ser herdadas por elementos filhos, aqui estão as propriedades de estilo mais comuns de serem herdadas por elementos filhos: 

            - cor
            - fonte
            - font-size

property: inital; Desativa a herança em uma propriedade, property: inherit; ativa a herança em uma propriedade

Display Rules

Dependendo do tipo de display de uma tag, ela pode ser incompativel com algumas propriedades, por exemplo: 

            Display-Inline - uma tag com display in-line é incompativel com largura, altura, margin, padding, a tag's de display in-line ocupam apenas a largura de seu conteúdo


            Display-block - Se o conteúdo de uma tag com display block exceder a largura da tag, o conteúdo transborda o tamanho da tag

CSS position's

static- posicionamento padrão, segueo fluxo normal do programa relative - Segue o fluxo normal do programa, mas pode quebrar o fluxo com propriedades especificas(right, top, left, buttom e etc) absolute - Se posiciona na tag root, ou se posiciona dentro de qualquer tag/box/element parent que não for static. fixed - Vai ter uma posição fixa, independente da posição do scroll, normalmente usado em navbar styck - Box/elemento/tag vai ficar fixo dentro de um elemento com scoll

Media Queries

Media querie é um recurso muitas vezes usado para tornar o site mais responsivo para todo tipo de tela. Exemplo de Media Querie:

@media screen and (max-width: xpx){} @media print and (min-width: xpx){} @media all and (min-width: xpx){}

É recomendado usar break-point's existentes para tornar a responsividade do site mais fluída

FlexBox

Display Flex é um display muito utilizado posicionar os elementos filhos da parent-box de forma organizada, todo tag com diplay flex é box-level, seus filhos ficam posicionados em row, com a largura maxima de seu conteúdo

        display:flex - Define que a parent-box é display flex
        
        flex-direction - Define a direção que as caixas filhas vão seguir, se for o padrão(row), a reta principal vai ser a horitzontal, agora se o flex-direction for column a reta principal é a vertical

        flex-wrap - Se as caixas filhas não couberem na reta principal faz a quebra de linha
        -wrap - faz a quebra de linha se não couber

        Flex-Flow: direction wrap - É uma shorthand para definir a direção das caixas filhas, e se vai ter quebra caso não caibam na reta principal

        flex-grow: x; - Distribui igualmente o espaço que sobrar como tamanho para as caixas filhas, ou seja, todas as caixas filhas vão crescer na mesma proporção baseada no tamanho/espaço que sobrar

        flex-basis: xpx - Define um tamanho minimo para as tags/caixas/elementos filhos

        flex-shrink  - Define a proporção de redução das caixas filhas

        flex: grow shrink basis - Short-hand para distribuir igualmente espaço que sobrar entre caixas filhas, para permitir que as caixas filhas fiquem menor que o basis, e defini um tamanho para as caixas filhas



   


        align-content - Alinha todas as linhas, ou seja alinha todo o conteúdo em uma posição da reta segundária

        justify-content - Usado para alinhar caixas filhas em uma posição da reta principal
                flex-start - Alinha no inicio da reta principal
                center - Alinha ao centro da reta principal
                flex-end - Alinha ao final da reta principal
                stretch - Estica as caixas filhas em toda a largura da reta principal
                space-around - Mesmo espaço apenas entre elementos
                space-betwen - Mesmo espaço para todos os elementos, sem o espaçamento inicial da reta principal
                space-evenly - Espaçamento é o mesmo para lugares preenchidos ou não


        align-items - Usado para alinhar a linha das caixas filhas em uma posição da reta secundária
                flex-start - Alinha no inicio da reta secundária
                center - Alinha ao centro da reta secundária
                flex-end - Alinha ao final da reta secundária
                stretch - Estica caixas filhas até a largura da reta secundária


        align-self - Alinha apenas um elemento da linha em uma posição da reta secundária, ao contrário do align-items que alinha uma linha de elementos/tags/box em uma posição da reta secundária

        order - Define a ordem de uma caixa filha dentro de uma flex-box

        Obs: É Possível fazer com que determinadas caixas filhas recebam o dobro de espaço em branco em relação as demais, ou diminuam com metade da proporção em relação as demais caixas

CSS GRID

É umd display muito usado para criação de layout's de páginas, ou para criação de tabelas, quando é precis que as caixas filhas se posicionem em muitas linhas ou colunas. Exemplos de propriedade: 

            display: grid;  - Define que a tag/elemento vai ter um display-grid
            grid-template-columns: size size... - Define quantas colunas a grid vai ter
                obs: É possivel nomear colunas e linhas exemplo
                    grid-template-columns: [coluna1] 1fr [coluna2] 1fr; - assim ao invés de usar o indice para indicar quando uma linha ou coluna começa, é possível usar o nome da linha/coluna para determinar onde uma tag/elemento começa e termina, o nome deve ser usado sem o colchetes

            grid-template-rows: size size... - Define quantas linhas a grid vai ter

            gap - Define espaçamento entre linhas e colunas

            minmax(min, max) - É usado em conjunto com grid-template-columns, ou grid-template-rows, para definir um tamanho minimo e maximo de um coluna ou linha

            auto-fill - preencher de acordo com o espaço disponivel 

            repeat(auto-fill, minmax(x, y)) Essa função vai tentar criar o maior número de colunas possiveis referênciando o tamanho da tela, ou seja vai criar colunas com tamanho minmax, de acordo com o tamanho da tela


            grid-colunm-start - Define em qual coluna uma tag/elemento vai começar
            grid-colunm-end  - Define em qual coluna uma tag/elemento vai terminar 
            
            grid-row-start - Define em qual linha uma tag/elemento vai começar
            grid-row-end - Define em qual linha um tag/elemento vai terminar

            grid-row: span x;| x / y  - Define quantas linhas uma tag/elemento pode ocupar 
            grid-colunm: span x;| x / y - Define quantas colunas uma tag/elemento pode ocupar

            grid-area: startrow / startcolunm / endrow / endcolunm - Determina o inicio e o fim das linhas e colunas de uma tag/elemento,

            grid-area: name - também pode ser usado para setar o areaname de uma tag/elemento
            
            grid-template-area: 'areaname areaname...' 'areaname...' values... - É usado para definir quantas colunas e em quais linhas uma tag/elemento vai se posicionar


            row-gap - Define o espaçamento entre linhas
            column-gap  - Define o espaçamento entre colunas

Text Property's

color - Define a cor do conteúdo background-color - Altera cor de fundo de uma tag font-size - Define o tamanho do texto font-weight - Define se o texto é negrito ou não font-style - Define se o texto é italico ou não font-family - Define a fonte de um texto letter-spacing - Define espaçamento entre as letras word-spacing - Define espaçamento entre as palavras line-height - Define altura da linha text-align - Define alinhamento do texto text-ident - Define a identação do texto direction - Define a direção do texto: -rtl - right to left direita pra esquerda -ltr - Left to right, esquerda para direita text-decoration: value [value]; - Propriedade para definir decoração do texto, sublinhado e etc. Obs: É possivel combinar valores de text-decoration text-shadow: offset-horizontal offset-vertical blur spread color text-transform: - Transforma conteúdo do texto para minúsculo/maiúsculo, capitalize e etc

value:

                -none - sem decoração nenhuma
                -underline - vai ter underline
                -overline - vai ter uma linha acima do conteúdo
                -line-trough - inseri uma linha atrávez do texto, ou seja vai riscar o texto

            overflow-x
            overflow-y
